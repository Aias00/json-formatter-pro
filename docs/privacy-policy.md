# Privacy Policy for JSON Formatter Pro

Last updated: February 27, 2026

JSON Formatter Pro processes JSON data locally in your browser.

## Data Collection and Use

- We do not collect personal information.
- We do not transmit your JSON content to external servers.
- We do not sell or share user data.
- We do not use analytics, advertising identifiers, or tracking beacons.

## Permissions

- `storage`: used only to save local preferences such as theme, indentation, and auto-format settings.
- `clipboardWrite`: used only when you explicitly click copy actions to copy output text.

## Data Storage

All extension settings are stored locally in your browser via Chrome extension storage.

## Third-Party Services

This extension does not use third-party backend services for JSON processing.

## Remote Code

All executable code is packaged with the extension. No remote scripts or remote WebAssembly are executed.
